`npm start` to run scripts.

POST FORMAT:
```json
{
      "first_name": "einsteinder",
      "last_name": "sparrow",
      "email": "einsteinder@gmail.com",
      "gender":"male",
      "ip_address":"127.0.0.1"

```

```json

      "last_name": "sparrow",

      "gender":"male",

```
